# Linee guida per contribuire

Il branch `main` deve contenere soltanto codice verificato.

Branch suggeriti:

```text
feature/compact-model
feature/threshold-model
feature/instance-generator
feature/greedy-heuristic
feature/local-search
feature/experiments
```

Messaggi di commit in italiano, brevi e descrittivi:

```text
Aggiunge la validazione delle istanze
Corregge il calcolo della capacità residua
Introduce il modello compatto
```

Regole principali:

- funzioni, classi e variabili in inglese;
- documentazione e commenti esplicativi in italiano;
- configurazioni fuori dal codice;
- risultati grezzi non modificati manualmente;
- nessuna credenziale o licenza locale pubblicata.
