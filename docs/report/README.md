# Relazione di studio

Questa cartella contiene la relazione estesa utilizzata per comprendere e preparare la presentazione del progetto.

## Documento principale

- [`CMMOFLP_Nuclear_Siting_Report.md`](CMMOFLP_Nuclear_Siting_Report.md) — descrizione completa del problema, delle motivazioni reali ed etiche, dei modelli, degli algoritmi, del benchmark e dei risultati.

La relazione ha finalità didattica e accompagna la documentazione tecnica modulare presente in `docs/`. Le informazioni numeriche derivano dai risultati pubblicati in `results/final/`.

## Percorso di studio consigliato

1. leggere le sezioni 1–5 per comprendere motivazione, significato etico, problema e modelli esatti;
2. studiare le sezioni 6–7 per le euristiche;
